# Route commerciale optimisée — Maximum de points AA

> Basé sur `AllianceAid_Guide.md` et les missions disponibles ci-dessus.
> Objectif : enchaîner les missions interstellaires (405 pts/mission) en groupant les pickups géographiquement.

## Principe clé

Les missions **interstellaires vers Levski** donnent **405 pts** contre **225 pts** pour les missions intra-Stanton. Il faut donc :
1. Privilégier systématiquement les missions Covalex (interstellaires)
2. Grouper les pickups sur le même trajet avant de sauter vers Nyx

### Boucle optimale disponible avec les missions actuelles

```

     Teasa Spaceport : charger 8 SCU filtres (Mission 9 → Levski)  │
                                                                    │
[2] Crusader (orbite)                                               │
     Seraphim Station : charger 6 SCU médical (Mission 6 → Levski) │ ALLER
                                                                    │
[3] microTech                                                       │
     NB Int. Spaceport : charger 8 SCU traitement (Mission 5 → Levski) │
                                                                    │
[4] ──── JUMP Nyx ──────────────────────────────────────────────────┘
     Levski : livrer les 3 missions (22 SCU total)
     = 1 215 points · ¤ 241 250
```

**Bilan aller :** 3 missions × 405 pts = **1 215 pts** · **¤ 241 250** · 22 SCU requis

### Retour Levski → Stanton (missions à vérifier sur place à Levski)

D'après le guide, les missions disponibles depuis Levski incluent :
- **Interstellar Filters** (Levski → Levski) : 405 pts · ¤ 93 500 · 8 SCU
- **Interstellar Research** (Levski → New Babbage) : 405 pts · ¤ 83 500 · 8 SCU
- **Interstellar Cryopods** (Levski → New Babbage) : 390 pts · ¤ 80 500 · 12 SCU

